# Output Levels

The robot console now hides normal messages and prints only suspicious states.

## NEAR SUSPICIOUS

Early signal. Something may be starting, but the system is not confident enough for a high alarm.

## HIGH SUSPICIOUS

Strong suspicious movement pattern. This should be treated as a serious alert for the demo.

## CRITICAL KIDNAPPING RISK

Very high risk signal that persisted over time. This is the closest safe wording to the user's requested “100% kidnapping detected.”

The system must not claim true certainty or intent. It should say **critical risk**, not **100% kidnapping**.
